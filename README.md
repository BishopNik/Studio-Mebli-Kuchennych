# FORMA — кухни на заказ

Адаптивная промостраница польской мастерской кухонь и корпусной мебели на заказ. Интерфейс и контент сайта полностью на польском языке.

## Запуск

Требуется [Node.js](https://nodejs.org/) версии 18 или новее. Установка зависимостей не нужна.

```bash
npm start
```

После запуска откройте адрес, указанный в терминале. По умолчанию это:

```text
http://localhost:4173
```

Если порт `4173` уже используется, сервер автоматически попробует `4174`, `4175` и следующие свободные порты.

Чтобы остановить сервер, нажмите `Ctrl+C`.

Также можно задать порт вручную:

```bash
PORT=8080 npm start
```

## Что реализовано

- премиальный адаптивный дизайн для компьютеров и телефонов;
- польская семантическая HTML-разметка и базовые SEO-метатеги;
- портфолио с фильтрацией по стилю;
- описание процесса работы и дополнительных мебельных услуг;
- интерактивный калькулятор ориентировочной стоимости кухни;
- отзывы, раскрывающийся FAQ и форма заявки с уведомлением;
- мобильное меню и фиксированная навигация;
- оптимизированные локальные изображения без внешнего фотохостинга;
- лёгкий статический Node-сервер без сторонних пакетов.

## Структура проекта

```text
.
├── assets/            # изображения и favicon
├── index.html         # разметка страницы
├── styles.css         # дизайн и адаптивные стили
├── script.js          # фильтры, калькулятор, FAQ и формы
├── server.mjs         # локальный статический сервер
└── package.json       # npm-команды и требования к Node.js
```

## Важно

Текущая версия — готовый frontend-прототип главной страницы. Форма показывает локальное подтверждение, но пока не отправляет данные на сервер. CMS, база данных, авторизация, очередь заказов, отправка email и генерация PDF из полного технического задания требуют отдельной backend-части.

## Развёртывание

Проект состоит из статических файлов, поэтому его можно разместить на Vercel, Netlify, Cloudflare Pages, GitHub Pages либо любом сервере с Nginx. Для статического хостинга публикуется корень проекта; `server.mjs` нужен только для локального запуска.
